PowerAPI-ruby
============
Library for fetching information from PowerSchool SISes.

Requirements
------------
* Ruby >= 2.0.0
* PowerSchool 8.x; PowerSchool >= 7.1.0

Usage
-----
You should use [RubyGems](https://rubygems.org/) to handle including/downloading
the library for you.

```
$ gem install powerapi
```
